Agent Modules

主页突出的是这个仓库相对普通聊天 Demo 的重点：本地模型、长期记忆、语音/视觉输入，以及可被桌面场景调用的 Agent 能力。

M

Memory

为对话、人物设定和长期偏好预留本地记忆链路，减少对云端状态的依赖。

V

Vision

通过视觉捕获和上下文感知，让 Agent 理解窗口、画面和操作环境。

S

Speech

支持语音识别、说话检测和语音合成，把陪伴式交互从文字扩展到实时对话。

P

Plugins

Providers、插件和服务运行时可以继续挂载新的工具、游戏和自动化能力。

Stage Experience

从网页舞台到桌面窗口，AIRI Gemma 的宣传页保留角色、设置和开发者工具的实际产品画面。

Desktop Stage

桌面端以独立窗口承载角色、对话和系统控制，适合本地常驻运行。

Settings Console

模型、Provider、语音、视觉、开发者工具都可以通过设置面板集中管理。

Local Stack

它不是单文件聊天壳，而是一套围绕本地模型和 AIRI 生态拆开的应用、包、服务和集成。

Gemma + Ollama

中文部署指南聚焦 Gemma 4 和 Ollama，方便普通 Windows 用户启动本地模型。

Stage Web / Pocket / Tamagotchi

网页、移动和桌面场景共用 AIRI 的舞台式交互语言。

xsAI Providers
